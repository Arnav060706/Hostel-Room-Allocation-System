# Hostel-Room-Allocation-System
A Hostel Room Allocation System in C that assigns hostel rooms to students based on preference and availability

The following content describes steps to execute the project

The files that have been attached in the zip file are as follows:
DSA-Mini-Project.pptx
hostel_app.exe
main.c
source.c
hostel.h

The following steps should be followed to compile the code and then run it: (For windows)
1)gcc main.c source.c -o hostel_app
This creates the hostel_app.exe file that is an executable that should be ran
2)./hostel_app.exe
This runs the executable file
